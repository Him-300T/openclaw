## 二、短期实施（1周内）

### 行动3：安装lossless-claw-enhanced插件
**目标**：改善中文对话上下文管理
**时间**：1小时
**步骤**：
1. 克隆仓库：`git clone https://github.com/win4r/lossless-claw-enhanced.git`
2. Link模式安装：`openclaw plugins install -l ./lossless-claw-enhanced`
3. 修改OpenClaw配置添加contextEngine slot
4. 重启Gateway：`openclaw gateway restart`
5. 验证安装：`openclaw plugins inspect lossless-claw`

**配置内容**：
```json
{
  "plugins": {
    "slots": {
      "contextEngine": "lossless-claw"
    },
    "entries": {
      "lossless-claw": {
        "enabled": true,
        "config": {
          "freshTailCount": 32,
          "contextThreshold": 0.75,
          "incrementalMaxDepth": -1,
          "ignoreSessionPatterns": [
            "agent:*:cron:**",
            "agent:*:subagent:**"
          ],
          "summaryModel": "anthropic/claude-haiku-4-5"
        }
      }
    }
  }
}
```

**验证步骤**：
1. 运行`openclaw plugins list`确认插件识别
2. 运行`openclaw plugins inspect lossless-claw`确认加载
3. 运行`openclaw doctor`健康检查
4. 测试长对话保持上下文能力

**产出**：
- 安装过程记录
- 配置备份文件
- 效果测试报告

**成功标准**：
- 插件成功安装并启用
- 中文对话体验明显改善
- 无兼容性问题或性能下降
