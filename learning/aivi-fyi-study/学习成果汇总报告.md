**理由**：
- 直接改善中文对话体验
- 解决当前可能存在的上下文管理问题
- 安装相对简单，风险低

**实施步骤**：
1. 克隆仓库：`git clone https://github.com/win4r/lossless-claw-enhanced.git`
2. link模式安装：`openclaw plugins install -l ./lossless-claw-enhanced`
3. 修改配置添加contextEngine slot
4. 重启Gateway验证

**预期效果**：
- 更准确的中文token估算
- 更合理的上下文压缩时机
- 长对话保持更好的一致性
